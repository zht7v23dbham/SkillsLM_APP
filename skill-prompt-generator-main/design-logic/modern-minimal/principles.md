# 现代简约风格 - 设计原则

## 风格定位

**核心情绪**：专业、高效、克制、现代
**目标受众**：成人、专业人士、商务人士
**适用场景**：科技产品、商务演示、企业宣传、专业服务

## 设计原则

### 1. 配色原则

**主色调选择**：
- 使用深色系或中性色（Dark or Neutral）
- 饱和度可高可低（根据子风格）
- 明度适中（避免过亮或过暗）

**常用色系**：
- 深蓝色系（Deep Blue）：#1F2937, #374151, #4B5563
- 紫色系（Purple）：#6366F1, #8B5CF6, #A78BFA
- 深绿色系（Dark Green）：#059669, #10B981, #34D399
- 中性灰系（Neutral Gray）：#1F2937, #374151, #6B7280

**配色规则**：
- 主色：深色或饱和色
- 辅色：与主色和谐（同色系或互补色）
- 背景：纯白#FFFFFF或极浅灰#F9FAFB
- 文字：深灰#1F2937（高对比）

### 2. 形状原则

**圆角设计**：
- 使用小到中等圆角（4-12px）
- 圆角营造现代感但不失专业
- 统一圆角规格（通常8px）

**避免**：
- ❌ 大圆角（>16px，显得可爱不专业）
- ❌ 完全直角（0px，显得过时）
- ❌ 圆形（过于柔和）
- ✅ 克制的圆角，几何感

### 3. 阴影原则

**细微阴影**：
- 使用非常软、非常细的阴影
- 格式：`0 Xpx Ypx rgba(R,G,B,opacity)`
- X值：0-4px（水平扩散小）
- Y值：2-8px（垂直扩散小）
- 透明度：0.04-0.08（非常微妙）

**效果**：
- 微妙的层次感
- 不明显的深度
- 保持平面化、克制

### 4. 字体原则

**字体选择**：
- 使用几何无衬线字体（Geometric Sans-serif）
- 例如：Inter, SF Pro, Roboto, system-ui

**字重**：
- 标题：600-700（Semi-Bold）
- 正文：400-500（Regular-Medium）

**行距**：
- 1.5-1.6倍（紧凑但不拥挤）
- 比温馨风格更紧凑

**字间距**：
- -0.01-0.02em（标准或稍紧）
- 专业、高效的感觉

### 5. 装饰原则

**装饰元素**：
- 几何线条（直线、折线）
- 点阵（dot grid）
- 渐变色块（subtle gradients）
- 图标（线性图标，outline style）

**使用规则**：
- 选择1-2个元素组合
- 透明度：0.05-0.10（非常微妙）
- 旋转角度：0°或45°、90°（几何规律）
- 对齐网格（grid-aligned）

**避免**：
- 过度装饰（>5%面积）
- 不规则形状（organic shapes）
- emoji、手绘风格
- 过于花哨的效果

### 6. 留白原则

**留白比例**：
- 整体留白：50-55%的画布
- 内容密集度：中等
- 元素间距：16-24px（8px的倍数）

**作用**：
- 营造专业、高效的感觉
- 让信息层次清晰
- 符合成人认知（信息密度适中）

### 7. 交互原则

**按钮/可点击元素**：
- 微妙的视觉反馈
- Hover状态：颜色加深5-10%
- 阴影增强：0 4px 12px（但仍克制）
- 过渡动画：150-200ms（快速）

**专业文案**：
- "了解更多"
- "立即开始"
- "查看详情"
- "联系我们"

## 视觉层次

### 主要元素（最突出）
- 大标题（字号大、字重重）
- 核心数据/内容
- 主要CTA

### 次要元素
- 副标题、标签
- 说明文字
- 图表、图标

### 辅助元素
- 日期、签名
- 背景装饰
- 细节说明

## 情绪传递

**通过设计传达**：
- 专业感（克制、几何）
- 信任感（稳定、一致）
- 现代感（简洁、高效）
- 高级感（留白、细节）

## 避免的陷阱

❌ 过于活泼（大圆角、鲜艳色）
❌ 过于装饰（>10%装饰面积）
❌ 过于立体（强阴影、3D效果）
❌ 过于复杂（信息过载）
❌ 情绪化色彩（全粉、全彩虹）
❌ 手绘风格（不专业）

## 示例组合

**典型现代简约风配色**：
```
主色：#6366F1（靛蓝）
辅色：#8B5CF6（紫色）
背景：#FFFFFF（纯白）
文字：#1F2937（深灰）
装饰：几何线条 opacity:0.08
```

**典型边框阴影**：
```
border-radius: 8px
box-shadow: 0 2px 8px rgba(0,0,0,0.06)
```

## 子风格

### 1. 科技未来（Tech Future）
- 主色：深蓝、紫色、青色
- 装饰：渐变、点阵
- 用途：科技产品、AI、软件

### 2. 专业商务（Business Pro）
- 主色：深蓝、深绿、中性灰
- 装饰：几何线条
- 用途：企业服务、B2B、咨询

### 3. 高级黑金（Premium Black-Gold）
- 主色：黑色#1A1A1A + 金色#D4AF37
- 装饰：细线、渐变
- 用途：奢侈品、高端服务
